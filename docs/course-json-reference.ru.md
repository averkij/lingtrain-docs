# Справочник JSON-формата курсов {#json-reference}

Этот справочник документирует JSON-схемы, используемые для контента курсов в Lingtrain. Эти схемы используются редактором юнитов, функциями импорта/экспорта и генерацией с помощью ИИ.

## Схема course-unit.v1 {#unit-schema}

Контент каждого юнита хранится как JSON-объект, соответствующий схеме `course-unit.v1`.

### Поля верхнего уровня {#unit-top-level}

| Поле | Тип | Обязательное | Описание |
|------|-----|-------------|----------|
| `schema_version` | `string` | Да | Должно быть `"course-unit.v1"` |
| `kind` | `string` | Да | Тип юнита: `lesson`, `reference`, `parallel_reading`, `drill`, `assessment`, `project` |
| `title` | `string` | Да | Отображаемое название юнита |
| `summary` | `string` | Нет | Краткое описание юнита |
| `learning_objectives` | `string[]` | Нет | Список учебных целей |
| `estimated_minutes` | `integer` | Нет | Примерное время прохождения (в минутах) |
| `content` | `Block[]` | Нет | Массив блоков контента (см. [Типы блоков](#block-types)) |
| `activities` | `Activity[]` | Нет | Массив интерактивных активностей (см. [Типы активностей](#activity-types)) |
| `completion_rule` | `CompletionRule` | Нет | Правило определения прохождения (см. [Правила завершения](#completion-rules)) |

**Лимит размера:** 256 КБ на payload юнита.

### Правила валидации {#unit-validation}

- `schema_version` должно быть точно `"course-unit.v1"`
- `kind` должно быть одним из шести допустимых значений
- `title` должно быть непустой строкой
- Каждый блок должен иметь уникальный `block_id` и допустимый `type`
- Каждая активность должна иметь уникальный `activity_id` и допустимый `type`
- Отсутствие `summary`, `learning_objectives` или `estimated_minutes` создаёт предупреждения (не ошибки)

## Типы блоков {#block-types}

Блоки контента -- строительные элементы юнита. Каждый блок имеет общие поля `block_id` и `type`, а также специфичные для типа поля.

### rich_text {#block-rich-text}

Форматированный текст с поддержкой Markdown.

| Поле | Тип | Обязательное | Описание |
|------|-----|-------------|----------|
| `block_id` | `string` | Да | Уникальный идентификатор блока |
| `type` | `string` | Да | Должно быть `"rich_text"` |
| `body` | `string` | Да | Текст в формате Markdown |

### callout {#block-callout}

Выделенный информационный блок с визуальным тоном.

| Поле | Тип | Обязательное | Описание |
|------|-----|-------------|----------|
| `block_id` | `string` | Да | Уникальный идентификатор блока |
| `type` | `string` | Да | Должно быть `"callout"` |
| `tone` | `string` | Да | Один из: `info`, `tip`, `warning`, `danger` |
| `title` | `string` | Нет | Необязательный заголовок выноски |
| `body` | `string` | Да | Текст выноски |

### term_grid {#block-term-grid}

Фильтруемая сетка терминов с определениями.

| Поле | Тип | Обязательное | Описание |
|------|-----|-------------|----------|
| `block_id` | `string` | Да | Уникальный идентификатор блока |
| `type` | `string` | Да | Должно быть `"term_grid"` |
| `terms` | `Term[]` | Да | Массив объектов терминов |

Каждый объект `Term`:

| Поле | Тип | Обязательное | Описание |
|------|-----|-------------|----------|
| `term` | `string` | Да | Термин или слово |
| `definition` | `string` | Да | Определение или перевод |

### dialogue {#block-dialogue}

Диалог в формате чат-пузырей.

| Поле | Тип | Обязательное | Описание |
|------|-----|-------------|----------|
| `block_id` | `string` | Да | Уникальный идентификатор блока |
| `type` | `string` | Да | Должно быть `"dialogue"` |
| `lines` | `Line[]` | Да | Массив реплик диалога |

Каждый объект `Line`:

| Поле | Тип | Обязательное | Описание |
|------|-----|-------------|----------|
| `speaker` | `string` | Да | Имя или идентификатор говорящего |
| `text` | `string` | Да | Реплика |

### code {#block-code}

Блок кода с подсветкой синтаксиса.

| Поле | Тип | Обязательное | Описание |
|------|-----|-------------|----------|
| `block_id` | `string` | Да | Уникальный идентификатор блока |
| `type` | `string` | Да | Должно быть `"code"` |
| `language` | `string` | Нет | Язык программирования для подсветки |
| `code` | `string` | Да | Содержимое кода |

### parallel_reader {#block-parallel-reader}

Параллельный двуязычный текст из снимка выравнивания.

| Поле | Тип | Обязательное | Описание |
|------|-----|-------------|----------|
| `block_id` | `string` | Да | Уникальный идентификатор блока |
| `type` | `string` | Да | Должно быть `"parallel_reader"` |
| `alignment_id` | `integer` | Нет | ID исходного выравнивания (разрешается при публикации) |
| `pairs` | `Pair[]` | Да | Массив пар предложений |

Каждый объект `Pair`:

| Поле | Тип | Обязательное | Описание |
|------|-----|-------------|----------|
| `from` | `string` | Да | Текст на исходном языке |
| `to` | `string` | Да | Текст на целевом языке |

## Типы активностей {#activity-types}

Активности -- интерактивные упражнения, оцениваемые серверным движком. Каждая активность имеет общие поля `activity_id` и `type`.

### multiple_choice {#activity-multiple-choice}

Вопрос с выбором одного или нескольких вариантов ответа.

| Поле | Тип | Обязательное | Описание |
|------|-----|-------------|----------|
| `activity_id` | `string` | Да | Уникальный идентификатор активности |
| `type` | `string` | Да | Должно быть `"multiple_choice"` |
| `prompt` | `string` | Да | Текст вопроса |
| `options` | `Option[]` | Да | Массив вариантов ответа |
| `multiple` | `boolean` | Нет | Разрешить множественный выбор (по умолчанию: `false`) |

Каждый объект `Option`:

| Поле | Тип | Обязательное | Описание |
|------|-----|-------------|----------|
| `id` | `string` | Да | Идентификатор варианта |
| `text` | `string` | Да | Текст варианта |
| `correct` | `boolean` | Да | Является ли вариант правильным |
| `explanation` | `string` | Нет | Объяснение, показываемое после блокировки попыток |

### match_pairs {#activity-match-pairs}

Упражнение на сопоставление пар.

| Поле | Тип | Обязательное | Описание |
|------|-----|-------------|----------|
| `activity_id` | `string` | Да | Уникальный идентификатор активности |
| `type` | `string` | Да | Должно быть `"match_pairs"` |
| `prompt` | `string` | Нет | Необязательная инструкция |
| `pairs` | `MatchPair[]` | Да | Массив пар для сопоставления |

Каждый объект `MatchPair`:

| Поле | Тип | Обязательное | Описание |
|------|-----|-------------|----------|
| `left` | `string` | Да | Элемент левой стороны |
| `right` | `string` | Да | Соответствующий элемент правой стороны |

### fill_in_blank {#activity-fill-in-blank}

Текст с пропусками для заполнения.

| Поле | Тип | Обязательное | Описание |
|------|-----|-------------|----------|
| `activity_id` | `string` | Да | Уникальный идентификатор активности |
| `type` | `string` | Да | Должно быть `"fill_in_blank"` |
| `prompt` | `string` | Нет | Необязательная инструкция |
| `segments` | `Segment[]` | Да | Массив сегментов текста и пропусков |

Каждый объект `Segment`:

| Поле | Тип | Обязательное | Описание |
|------|-----|-------------|----------|
| `type` | `string` | Да | `"text"` или `"gap"` |
| `value` | `string` | Да | Текст (для `text`) или правильный ответ (для `gap`) |
| `alternatives` | `string[]` | Нет | Дополнительные допустимые ответы (только для `gap`) |

### order_sequence {#activity-order-sequence}

Расстановка элементов в правильном порядке.

| Поле | Тип | Обязательное | Описание |
|------|-----|-------------|----------|
| `activity_id` | `string` | Да | Уникальный идентификатор активности |
| `type` | `string` | Да | Должно быть `"order_sequence"` |
| `prompt` | `string` | Нет | Необязательная инструкция |
| `items` | `string[]` | Да | Элементы в правильном порядке (перемешиваются при отображении) |

### flashcards {#activity-flashcards}

Двусторонние карточки с 3D-анимацией переворота.

| Поле | Тип | Обязательное | Описание |
|------|-----|-------------|----------|
| `activity_id` | `string` | Да | Уникальный идентификатор активности |
| `type` | `string` | Да | Должно быть `"flashcards"` |
| `prompt` | `string` | Нет | Необязательная инструкция |
| `cards` | `Card[]` | Да | Массив объектов карточек |

Каждый объект `Card`:

| Поле | Тип | Обязательное | Описание |
|------|-----|-------------|----------|
| `front` | `string` | Да | Текст лицевой стороны (вопрос/термин) |
| `back` | `string` | Да | Текст обратной стороны (ответ/определение) |

### short_text {#activity-short-text}

Свободный ввод текста с проверкой по допустимым ответам.

| Поле | Тип | Обязательное | Описание |
|------|-----|-------------|----------|
| `activity_id` | `string` | Да | Уникальный идентификатор активности |
| `type` | `string` | Да | Должно быть `"short_text"` |
| `prompt` | `string` | Да | Вопрос или инструкция |
| `accepted_answers` | `string[]` | Да | Список правильных ответов (регистронезависимое сравнение) |
| `explanation` | `string` | Нет | Объяснение, показываемое после блокировки попыток |

## Правила завершения {#completion-rules}

Поле `completion_rule` определяет, как сервер определяет прохождение юнита учеником.

| Правило | Поля | Описание |
|---------|------|----------|
| `all_required` | `{ "type": "all_required" }` | Ученик должен выполнить каждую активность |
| `score_threshold` | `{ "type": "score_threshold", "threshold": 0.8 }` | Ученик должен набрать указанный балл (0.0--1.0) |
| `mixed` | `{ "type": "mixed", "threshold": 0.8 }` | Ученик должен выполнить все активности И набрать пороговый балл |

Если `completion_rule` не указано, по умолчанию используется `all_required`.

## Схема course-bundle.v1 {#bundle-schema}

Пакет курса объединяет весь курс (с модулями и юнитами) в один JSON-файл для импорта или генерации с помощью ИИ.

### Поля верхнего уровня {#bundle-top-level}

| Поле | Тип | Обязательное | Описание |
|------|-----|-------------|----------|
| `schema_version` | `string` | Да | Должно быть `"course-bundle.v1"` |
| `title_en` | `string` | Нет | Название курса на английском |
| `title_ru` | `string` | Нет | Название курса на русском |
| `summary_en` | `string` | Нет | Описание курса на английском |
| `summary_ru` | `string` | Нет | Описание курса на русском |
| `domain` | `string` | Да | Одно из: `language-learning`, `programming`, `general`, `writing` |
| `difficulty` | `string` | Да | Одно из: `beginner`, `intermediate`, `advanced` |
| `tags` | `string[]` | Нет | Теги курса для фильтрации в каталоге |
| `modules` | `Module[]` | Да | Массив объектов модулей |

Хотя бы одно из `title_en` или `title_ru` должно быть указано.

### Объект модуля {#bundle-module}

| Поле | Тип | Обязательное | Описание |
|------|-----|-------------|----------|
| `title` | `string` | Да | Название модуля |
| `units` | `Unit[]` | Да | Массив объектов юнитов в этом модуле |

### Объект юнита (в пакете) {#bundle-unit}

| Поле | Тип | Обязательное | Описание |
|------|-----|-------------|----------|
| `title` | `string` | Да | Название юнита |
| `kind` | `string` | Да | Тип юнита (те же значения, что и в `course-unit.v1`) |
| `estimated_minutes` | `integer` | Нет | Примерное время прохождения |
| `unit_spec` | `object` | Да | Полный payload формата `course-unit.v1` |

## Примеры {#examples}

### Юнит-урок {#example-lesson}

Обучающий юнит с текстом, выноской и упражнением на выбор ответа.

```json
{
  "schema_version": "course-unit.v1",
  "kind": "lesson",
  "title": "Введение в настоящее время",
  "summary": "Основы спряжения глаголов в настоящем времени в английском языке.",
  "learning_objectives": [
    "Определять глаголы в настоящем времени",
    "Спрягать правильные глаголы в настоящем времени"
  ],
  "estimated_minutes": 15,
  "content": [
    {
      "block_id": "b1",
      "type": "rich_text",
      "body": "**Настоящее время** (Present Tense) используется для описания действий, происходящих сейчас, или привычных действий.\n\nДля правильных глаголов добавьте **-s** или **-es** к основной форме для третьего лица единственного числа."
    },
    {
      "block_id": "b2",
      "type": "callout",
      "tone": "tip",
      "title": "Запомните",
      "body": "Глаголы, оканчивающиеся на -ch, -sh, -ss, -x или -z, получают **-es** вместо **-s**."
    }
  ],
  "activities": [
    {
      "activity_id": "a1",
      "type": "multiple_choice",
      "prompt": "Какое предложение использует правильную форму настоящего времени?",
      "options": [
        { "id": "o1", "text": "She walk to school.", "correct": false },
        { "id": "o2", "text": "She walks to school.", "correct": true, "explanation": "Третье лицо единственного числа требует окончание -s." },
        { "id": "o3", "text": "She walking to school.", "correct": false }
      ]
    }
  ],
  "completion_rule": { "type": "all_required" }
}
```

### Юнит-справочник {#example-reference}

Словарный справочник с сеткой терминов.

```json
{
  "schema_version": "course-unit.v1",
  "kind": "reference",
  "title": "Кухонная лексика",
  "summary": "Основные кухонные термины на английском и русском.",
  "estimated_minutes": 5,
  "content": [
    {
      "block_id": "b1",
      "type": "term_grid",
      "terms": [
        { "term": "knife", "definition": "Острый инструмент для нарезки" },
        { "term": "frying pan", "definition": "Плоская сковорода для жарки" },
        { "term": "oven", "definition": "Закрытая камера для выпечки" },
        { "term": "cutting board", "definition": "Плоская поверхность для нарезки продуктов" }
      ]
    }
  ],
  "activities": [],
  "completion_rule": { "type": "all_required" }
}
```

### Юнит параллельного чтения {#example-parallel-reading}

Параллельный двуязычный текст для практики чтения.

```json
{
  "schema_version": "course-unit.v1",
  "kind": "parallel_reading",
  "title": "Маленький принц, Глава 1",
  "summary": "Прочитайте начало Маленького принца на английском и русском.",
  "estimated_minutes": 10,
  "content": [
    {
      "block_id": "b1",
      "type": "parallel_reader",
      "pairs": [
        { "from": "Once when I was six years old I saw a magnificent picture in a book.", "to": "Когда мне было шесть лет, я увидел однажды удивительную картинку в книге." },
        { "from": "It was a picture of a boa constrictor in the act of swallowing an animal.", "to": "На картинке огромная змея -- удав -- глотала хищного зверя." }
      ]
    }
  ],
  "activities": [],
  "completion_rule": { "type": "all_required" }
}
```

### Юнит-практика {#example-drill}

Практический юнит с интерактивными упражнениями.

```json
{
  "schema_version": "course-unit.v1",
  "kind": "drill",
  "title": "Практика порядка слов",
  "summary": "Практикуйте расстановку слов в правильные английские предложения.",
  "estimated_minutes": 10,
  "content": [
    {
      "block_id": "b1",
      "type": "rich_text",
      "body": "Расставьте слова в правильном порядке, чтобы составить грамматически верное предложение."
    }
  ],
  "activities": [
    {
      "activity_id": "a1",
      "type": "order_sequence",
      "prompt": "Расставьте слова по порядку:",
      "items": ["The", "cat", "sat", "on", "the", "mat"]
    },
    {
      "activity_id": "a2",
      "type": "fill_in_blank",
      "prompt": "Заполните предложения:",
      "segments": [
        { "type": "text", "value": "She " },
        { "type": "gap", "value": "goes", "alternatives": ["goes"] },
        { "type": "text", "value": " to the store every " },
        { "type": "gap", "value": "morning", "alternatives": ["morning", "day"] },
        { "type": "text", "value": "." }
      ]
    }
  ],
  "completion_rule": { "type": "all_required" }
}
```

### Юнит-тест {#example-assessment}

Оцениваемый тест с пороговым баллом и несколькими типами активностей.

```json
{
  "schema_version": "course-unit.v1",
  "kind": "assessment",
  "title": "Тест: настоящее время",
  "summary": "Проверьте знания форм глаголов в настоящем времени.",
  "estimated_minutes": 15,
  "content": [
    {
      "block_id": "b1",
      "type": "callout",
      "tone": "info",
      "body": "У вас 3 попытки на каждый вопрос. Для прохождения нужно набрать 80%."
    }
  ],
  "activities": [
    {
      "activity_id": "a1",
      "type": "multiple_choice",
      "prompt": "Выберите ВСЕ правильные формы настоящего времени:",
      "multiple": true,
      "options": [
        { "id": "o1", "text": "runs", "correct": true },
        { "id": "o2", "text": "ran", "correct": false },
        { "id": "o3", "text": "eats", "correct": true },
        { "id": "o4", "text": "eaten", "correct": false }
      ]
    },
    {
      "activity_id": "a2",
      "type": "match_pairs",
      "prompt": "Сопоставьте глагол с его формой третьего лица:",
      "pairs": [
        { "left": "go", "right": "goes" },
        { "left": "have", "right": "has" },
        { "left": "do", "right": "does" },
        { "left": "be", "right": "is" }
      ]
    },
    {
      "activity_id": "a3",
      "type": "short_text",
      "prompt": "Какова форма третьего лица единственного числа глагола 'fly'?",
      "accepted_answers": ["flies"],
      "explanation": "Глаголы на согласная + y меняют окончание на -ies: fly -> flies."
    }
  ],
  "completion_rule": { "type": "score_threshold", "threshold": 0.8 }
}
```

### Юнит-проект {#example-project}

Открытое задание с инструкциями и карточками для справки.

```json
{
  "schema_version": "course-unit.v1",
  "kind": "project",
  "title": "Написать рассказ",
  "summary": "Примените изученное, написав короткий рассказ в настоящем времени.",
  "learning_objectives": [
    "Применить настоящее время в творческом письме",
    "Использовать лексику из этого модуля"
  ],
  "estimated_minutes": 30,
```
