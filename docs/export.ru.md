# Экспорт {#export}

Вкладка **Создание** позволяет просматривать и экспортировать выровненные тексты в различных форматах. Она доступна, когда у вас есть хотя бы один обработанный батч в выравнивании.

## Настройки экспорта {#export-settings}

![Настройки экспорта](img/create-export.en.png)

Настройте результат с помощью следующих параметров:

- **Структура абзацев** — использовать разбивку на абзацы из исходного («из») или целевого («в») текста. Это определяет, как предложения группируются в абзацы в экспортированном результате.
- **Порядок языков** — какой язык отображается слева (первым) в параллельном виде.
- **Стиль подсветки** — визуальное оформление параллельной книги:
  - **Simple** — без фоновой подсветки
  - **Pastel Fill** — сплошной пастельный фон для каждого языка
  - **Pastel Start** — градиентный фон с затуханием

## Предпросмотр {#preview}

Нажмите **«Сгенерировать предпросмотр»**, чтобы увидеть живой предпросмотр параллельной книги с текущими настройками. Предпросмотр показывает, как будет выглядеть финальная HTML-книга, включая структуру абзацев, порядок языков и стиль подсветки.

![Предпросмотр книги](img/create-preview.en.png)

Настраивайте параметры и пересоздавайте предпросмотр, пока результат вас не устроит.

## Форматы скачивания {#download}

![Форматы скачивания](img/create-download.en.png)

Доступны следующие форматы экспорта:

| Формат | Описание |
|--------|----------|
| **HTML-книга** | Параллельная книга с текстом бок о бок, оформленная и готовая к чтению в любом браузере. Включает титульную страницу и заголовки глав из тегов разметки. |
| **TMX-корпус** | Формат Translation Memory eXchange — стандартный формат для CAT-систем (memoQ, SDL Trados, OmegaT). Содержит выровненные пары предложений с метаданными языков. |
| **Корпус предложений** | Текстовые файлы с одним выровненным предложением на строку. Доступны отдельно для исходного и целевого языков. Полезны для NLP-исследований и обучения моделей машинного перевода. |
| **Корпус абзацев** | Текстовые файлы, сгруппированные по абзацам на основе выбранной структуры абзацев. Доступны для обоих языков. |
| **Структурированные форматы** | XML и JSON представления структуры выровненной книги. Подходят для пользовательских конвейеров обработки и анализа данных. |
| **База выравнивания** | Формат Lingtrain `.lt` — полная база данных выравнивания, включая все метаданные, эмбеддинги и историю правок. Используйте для резервного копирования или повторного импорта в Lingtrain. |

Нажмите кнопку скачивания рядом с любым форматом, чтобы сгенерировать и сохранить файл.
