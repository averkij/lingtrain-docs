# Подстрочник {#proxy}

Прокси-документы улучшают качество выравнивания для языковых пар, где модель эмбеддингов имеет ограниченные обучающие данные.

## Что такое прокси-документы? {#what-is-proxy}

Прокси-документ — это **машинный перевод** разбитого текста, который служит посредником при выравнивании. Прокси-тексты полезны, когда модель эмбеддингов имеет ограниченные обучающие данные для данной языковой пары — например, при выравнивании башкирского, чувашского или других малоресурсных языков.

Вместо прямого сравнения оригинальных предложений выравниватель сравнивает прокси-переводы, которые обычно сделаны на хорошо поддерживаемом языке, таком как английский или русский. Оригинальные тексты сохраняются в финальном результате — прокси используется только на этапе сопоставления.

## Когда использовать прокси {#when-to-use}

Используйте прокси-документы, когда:

- Один или оба языка являются **малоресурсными** (ограниченное представление в модели эмбеддингов)
- Прямое выравнивание даёт **низкое качество** — много конфликтов, сломанная диагональ на визуализации
- Языковая пара **дистантна** (например, японский–финский), и модель плохо справляется с межъязыковым сходством

Прокси не нужен для хорошо поддерживаемых языковых пар, таких как английский–русский, английский–немецкий или английский–французский — модели эмбеддингов по умолчанию хорошо справляются с ними.

## Создание прокси-документа {#creating-proxy}

1. **Загрузите и разбейте** исходный текст на вкладке [Тексты](uploading.ru.md)
2. **Скачайте разбитый текст** — нажмите кнопку скачивания в панели предпросмотра предложений. Это создаст файл `.txt` с одним предложением на строку
3. **Переведите его** с помощью любого сервиса машинного перевода (Google Translate, DeepL, Яндекс Переводчик и т.д.)
4. **Загрузите перевод** как прокси-документ в разделе «Прокси» на странице деталей выравнивания

![Раздел прокси-документов](img/proxy-documents.en.png)

Прокси должен содержать **точно такое же количество строк**, как и оригинальный разбитый текст. Каждая строка прокси соответствует предложению на той же позиции в оригинале.

## Загрузка прокси {#uploading-proxy}

На странице деталей выравнивания раздел **Прокси** содержит области загрузки для обоих языков — исходного («из») и целевого («в»).

![Обзор раздела прокси](img/proxy-empty.en.png)

Для каждого языка:

- **Перетащите** файл `.txt` в область загрузки или нажмите для выбора файла
- После загрузки статус меняется на **«Загружен»** с отображением имени файла
- Используйте кнопку **скачивания** для получения загруженного прокси
- Используйте кнопку **удаления** для его удаления
- Используйте кнопку **открытия** для просмотра текста прокси в новой вкладке

## Использование прокси при выравнивании {#using-proxy}

После загрузки прокси-документов включите их в [настройках выравнивания](alignment.ru.md#settings):

- **Использовать прокси (из)** — использовать прокси для исходного языка
- **Использовать прокси (в)** — использовать прокси для целевого языка

Можно включить прокси для одной или обеих сторон. При включении выравниватель вычисляет эмбеддинги из текста прокси вместо оригинала, а затем применяет полученное отображение к оригинальным предложениям.

## Прокси в редакторе {#proxy-in-editor}

Когда прокси-документы загружены, редактор может отображать прокси-переводы в виде **подстрочных аннотаций** под каждым предложением. Это помогает проверять качество выравнивания, даже если вы не владеете одним из языков.

![Редактор с подстрочными аннотациями](img/editor-subscriptions.en.png)

Переключите видимость подстрочника с помощью переключателя **«Показать подстрочник»** в заголовке панели конфликтов.
