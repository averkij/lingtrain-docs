# Lingtrain для издателей и преподавателей {#use-cases-publishers}

Lingtrain Aligner позволяет издателям создавать профессиональные двуязычные издания, а преподавателям — готовить качественные двуязычные учебные материалы. Экспорт в формате HTML-книги предоставляет настраиваемое форматирование, варианты структуры абзацев и несколько стилей выделения, подходящих для публикации и использования в учебных целях.

## Для издателей: двуязычные издания {#publishers}

### Почему двуязычные книги востребованы {#why-bilingual}

Двуязычные издания обслуживают несколько рынков:

- **Изучающие языки** на всех уровнях, желающие читать на целевом языке с опорой на родной
- **Носители языка в диаспоре**, желающие восстановить связь с языком семьи через литературу
- **Любители сравнительного литературоведения**, которым интересно наблюдать, как перевод трансформирует текст
- **Академические читатели**, которым нужно обращаться и к оригиналу, и к переводу
- **Экспатриантские сообщества**, поддерживающие связь с родной культурой за рубежом

Рынок двуязычных книг растёт по мере увеличения глобальной мобильности и популяризации изучения языков. Lingtrain обеспечивает инфраструктуру выравнивания, делающую двуязычное книгоиздание практичным в масштабе.

### Создание двуязычного издания {#creating-bilingual}

Рабочий процесс для издателей:

1. **Получите права** на исходный текст и перевод (или используйте тексты, находящиеся в общественном достоянии)
2. **Подготовьте чистые текстовые файлы** на обоих языках — удалите номера страниц, колонтитулы и издательские метаданные
3. **Разметьте структурные элементы** — используйте теги разметки Lingtrain для заголовков глав (`%%%%%h1.`), имён авторов (`%%%%%author.`) и других структурных маркеров
4. **Загрузите и выровняйте** в Lingtrain по стандартному рабочему процессу
5. **Проверьте качество выравнивания** — для публикационного качества каждая пара предложений должна быть верифицирована
6. **Настройте параметры экспорта** — выберите структуру абзацев, порядок языков и стиль выделения
7. **Экспортируйте и интегрируйте** — скачайте HTML-книгу и адаптируйте для вашего издательского процесса

### Варианты структуры абзацев {#paragraph-structure}

Настройка структуры абзацев определяет, как разрывы абзацев отображаются в экспортированной книге:

- **Из оригинала** — разрывы абзацев следуют структуре исходного текста. Лучший выбор, когда важен ритм абзацев оригинала.
- **Из перевода** — разрывы абзацев следуют структуре перевода. Полезно, когда переводчик реструктурировал абзацы для лучшего потока на целевом языке.

Выбирайте вариант, создающий наиболее естественное чтение для вашей основной аудитории.

### Стили выделения {#highlight-styles}

Lingtrain предлагает несколько стилей выделения для различения двух языков в HTML-книге:

**Простой (Simple)** — минимальное оформление с чистой типографикой. Языки различаются через тонкие визуальные подсказки (отступы, насыщенность шрифта). Лучше всего для читателей, предпочитающих чистый, книжный вид.

**Пастельная заливка (Pastel Fill)** — предложения на каждом языке имеют светлый фоновый цвет (например, бледно-голубой для оригинала, бледно-персиковый для перевода). Позволяет легко определить, к какому языку относится каждое предложение, даже при быстром просмотре.

**Пастельное начало (Pastel Start)** — выделяется только начало каждого предложения цветным маркером. Компромисс между Simple и Pastel Fill — создаёт визуальный якорь без окрашивания целых предложений.

Выбирайте стиль, соответствующий визуальному образу вашего издания и предпочтениям читателей. Для серьёзных литературных публикаций обычно лучше Simple. Для учебных материалов Pastel Fill обеспечивает наиболее чёткое визуальное разделение.

### Порядок языков {#language-order}

Настройте, какой язык отображается первым (слева/сверху) в параллельном виде:

- Для **изучающих язык** поставьте изучаемый язык первым — читатель встречает его первым и может проверить перевод ниже
- Для **справочных изданий** поставьте первым язык оригинала — это традиционное соглашение для научных двуязычных изданий
- Для изданий **культурного обмена**, ориентированных на диаспорные сообщества, поставьте первым язык наследия

## Для преподавателей: учебные материалы {#educators}

### Двуязычные материалы для чтения {#reading-materials}

Преподаватели могут использовать Lingtrain для создания адаптированных двуязычных материалов:

- **Книги для чтения по уровням** — выберите тексты, соответствующие уровню каждого студента, и создайте параллельные версии
- **Литературные антологии** — составьте подборки рассказов или отрывков в параллельном формате для уроков литературы
- **Предметные материалы** — создайте двуязычные версии текстов по естественным наукам, истории или обществознанию для программ двуязычного обучения
- **Культурология** — сопоставьте оригинальные тексты с переводами для сравнительного культурного анализа

### Применение в классе {#classroom}

**Упражнения на сравнение переводов:** выровняйте один исходный текст с несколькими переводами (разных переводчиков). Студенты сравнивают переводческие решения и обсуждают стратегии.

**Лексика в контексте:** создавайте параллельные тексты, сфокусированные на определённых лексических областях. Студенты изучают новые слова в аутентичных контекстах, а не в изолированных списках.

**Грамматический анализ:** выровненные предложения позволяют студентам наблюдать, как грамматические конструкции обрабатываются по-разному в разных языках. Предложение с сослагательным наклонением во французском рядом с английским эквивалентом иллюстрирует концепцию через реальные примеры.

**Понимание прочитанного:** студенты сначала читают версию на изучаемом языке, затем проверяют понимание по оригиналу. Это увлекательнее традиционных упражнений на понимание, поскольку они работают с настоящей литературой.

### Подготовка учебных материалов {#preparing-materials}

Советы для преподавателей:

1. **Начинайте с коротких текстов.** Одного рассказа или статьи достаточно для урока или домашнего задания. Полные романы — для программ расширенного чтения.

2. **Выбирайте тексты с педагогической ценностью.** Учитывайте не только языковой уровень, но и тематическое содержание — тексты, порождающие дискуссию, ценнее в классе.

3. **Отмечайте ключевую лексику.** Хотя в Lingtrain нет встроенной аннотации словаря, вы можете постобработать HTML-вывод, выделив целевую лексику.

4. **Создавайте ключи ответов.** Экспортируйте выравнивание как корпус предложений и используйте параллельные файлы для составления словарных списков, вопросов на понимание и ключей ответов.

5. **Думайте о доступности.** Формат HTML доступен в любом браузере, на любом устройстве. Студентам не нужно специальное программное обеспечение для чтения параллельной книги.

### Оценивание и тестирование {#assessment}

Выровненные тексты могут поддержать оценивание:

- Создавайте тесты с пропусками (cloze), удаляя слова из стороны целевого языка — студенты заполняют пробелы, опираясь на оригинал
- Используйте выровненные фрагменты для переводческих упражнений — студенты переводят отрывок и сравнивают с выровненным профессиональным переводом
- Составляйте тесты на понимание прочитанного, где студенты должны найти соответствующий фрагмент на другом языке

## Настройка HTML-вывода {#customizing-html}

### Варианты постобработки {#post-processing}

HTML-книгу, экспортированную из Lingtrain, можно дополнительно настроить:

- **CSS-стилизация** — изменение цветов, шрифтов, отступов и компоновки в соответствии с вашим фирменным стилем
- **Форматирование для печати** — добавление CSS print-стилей для создания двуязычных книг, готовых к печати
- **Интерактивные функции** — добавление JavaScript для показа перевода по клику, выделения лексики или воспроизведения аудио
- **Конвертация в электронные книги** — преобразование HTML в EPUB или MOBI для распространения на электронных читалках

### Серийное производство {#bulk-production}

Для издателей, выпускающих несколько двуязычных книг:

- Установите стандартный рабочий процесс Lingtrain с единообразными параметрами
- Создайте CSS-шаблоны для единообразного визуального стиля серии
- Используйте экспорт базы данных выравнивания (`.lt`) для резервного копирования и повторной обработки
- Рассмотрите библиотеку Python для пакетной обработки нескольких текстов
